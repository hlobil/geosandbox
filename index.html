Welcome to the Geosandbox!

The geosandbox is a little playground for people wanting to write a bit of code and test out some of the more common web libraries for making maps and visualizing geodata. Often times when I'm doing geovisualization, I want to quickly test out a function here and there and thought it might be handy to have a few examples to play with. I started this project during my Mozilla Science Fellowship between 2015-2016 and am slowly but surely trying to keep this thing alive. In the future, I'd like to make it easier to contribute examples, but for now, I hope this helps!
